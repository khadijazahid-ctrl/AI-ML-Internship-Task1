# AI-ML-Internship-Task1

This repository contains tasks completed as part of an AI/ML Internship.

## Task 1: Exploring and Visualizing the Iris Dataset

### Objective

Learn how to load, inspect, and visualize a dataset using Pandas, Matplotlib, and Seaborn.

### Dataset Used

* Iris Dataset

### Libraries Used

* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

### Steps Performed

1. Loaded the Iris dataset.
2. Inspected the dataset using:

   * head()
   * info()
   * describe()
3. Visualized data using:

   * Scatter Plot
   * Histograms
   * Box Plots
4. Analyzed feature distributions and relationships.

### Results and Findings

* The dataset contains 150 samples and 5 columns.
* Three flower species are included:

  * Setosa
  * Versicolor
  * Virginica
* Petal measurements provide clear separation among species.
* Visualizations helped identify data distributions and potential outliers.
